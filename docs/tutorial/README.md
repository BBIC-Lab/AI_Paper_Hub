# 使用教程

这页只保留教程导航，三个入口分别对应一条完整操作链路。建议按顺序阅读；如果你已经完成基础部署，也可以直接跳到第二步修改订阅和查询。

## 推荐阅读路径

1. [**从零开始配置**](/tutorial/quick-start)：适合第一次使用本项目，目标是把 GitHub 仓库、Actions、Pages、模型和基础订阅全部连起来，最终能手动跑出第一份日报。

2. [**订阅与查询配置**](/tutorial/configuration)：适合已经能打开网页或运行工作流的用户，目标是把研究方向拆成可维护的订阅与查询，并掌握精读区、速读区和每日上限的配置方式。

3. [**推荐链路说明**](/tutorial/workflow)：适合想理解结果为什么被推荐、如何排查漏选或误选、以及怎样控制成本的用户，目标是看懂从论文抓取到网页展示的完整链路。

## 你会得到什么

- 一个可以每天自动更新的 AI 论文阅读站点。
- 一套按研究主题组织的订阅和查询配置。
- 一份能解释推荐结果来源、排序和分区逻辑的排查地图。
