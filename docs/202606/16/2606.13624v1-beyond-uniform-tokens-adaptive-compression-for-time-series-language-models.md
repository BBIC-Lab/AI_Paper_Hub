## 摘要
大型语言模型（LLMs）通过共享令牌接口联合建模数值观测与文本上下文，实现了时间序列（TS）分析。然而，TS令牌和提示令牌呈现出根本不同的信息结构，使得统一令牌处理效率低下。在本文中，我们从非对称令牌的角度研究TS语言建模中的令牌效率。我们发现TS令牌具有高度不均匀的频谱贡献，其中许多令牌共享冗余频率模式，而一小部分保留了关键的时间证据。我们还观察到，提示令牌的影响随模型深度而衰减，表明在所有层中完全保留提示是不必要的。基于这些发现，我们开发了一个自适应令牌预算框架，通过频域结构压缩TS令牌，并逐层逐步减少提示令牌。在预测、分类、填补和异常检测任务上的实验表明，推理速度提升高达7.68倍，并在78%的评估设置中实现了性能增益，证明了非对称令牌压缩对于可扩展的TS基础模型的有效性。

## Abstract
Large language models (LLMs) have enabled time series (TS) analysis by jointly modeling numerical observations and textual context through a shared token interface. However, TS tokens and prompt tokens exhibit fundamentally different information structures, making uniform token processing inefficient. In this paper, we study token efficiency in TS language modeling from an asymmetric-token perspective. We show that TS tokens have highly uneven spectral contributions, where many tokens share redundant frequency patterns while a small subset preserves critical temporal evidence. We also observe that prompt-token influence attenuates with model depth, suggesting that full prompt retention across all layers is unnecessary. Based on these findings, we develop an adaptive token budgeting framework that compresses TS tokens via frequency-domain structure and progressively reduces prompt tokens across layers. Experiments across forecasting, classification, imputation, and anomaly detection demonstrate up to \textit{\textbf{7.68$\times$}} inference acceleration and performance gains in \textit{\textbf{78\%}} of evaluated settings, showing the effectiveness of asymmetric token compression for scalable TS foundation models.

## 1. 论文的核心问题与整体含义

- **核心问题**：在“大型语言模型+时间序列”的多模态框架下，TS（时间序列）令牌和文本提示令牌被统一处理为相同粒度的离散令牌，忽视了二者在信息结构和冗余特性上的根本差异，导致计算效率低下。  
- **研究动机**：
  - 时间序列数据在采集过程中已不可逆地丢失部分语义，通常需要大量文本提示以激活LLM的先验知识，这进一步增大了令牌数量和计算开销。
  - TS令牌内部存在大量频域冗余，许多令牌携带相似频谱特征，而仅少数令牌承载关键时间证据。
  - 提示令牌对TS表征的影响随着模型层深增加而迅速衰减，深层保留全部提示令牌不再必要。
- **整体含义**：作者提出一种**非对称令牌压缩视角**——对TS令牌和提示令牌采用不同策略进行解耦压缩，在保持或提升性能的同时，大幅降低计算开销。

## 2. 方法论

### 2.1 核心思想
- **TS令牌**在频域表现出稀疏的强信息集中性，可在频域进行自适应融合以去除冗余。
- **提示令牌**的影响随模型深度呈“金字塔衰减”效应，可在浅层逐步压缩，深层几乎可完全移除。

### 2.2 关键技术细节

**TS令牌频域合并（TS Token Merging）**  
1. 将TS令牌序列沿令牌轴做离散傅里叶变换（DFT），得到频域表示 $\{F_k\}$。  
2. 对每个频率分量 $k$，根据幅值累计超过总能量 95% 的原则筛选出活跃令牌集 $I_{b,k}$。  
3. 引入基于 Whittle 似然和 Fisher 信息 $s_{b,i,k}$ 的**频谱可辨识性度量**，量化每个令牌对该频率分量结构识别的贡献，并聚合为得分 $S_{b,k}$。  
4. 通过 Sigmoid 映射将得分转化为合并强度 $r_{b,k}$，自适应确定各频率分量的合并预算 $m_{b,k} = \lceil (1 - r_{b,k}) \cdot n_{b,k} \rceil$。  
5. 在活跃令牌集内部建立局部亲和图（邻域约束 $|i-j| < r_{\text{loc}}$），利用余弦相似度构建亲和矩阵，按预算对局部组内令牌做加权合并，得到合并后的频域表示 $\tilde{F}_{u,k}$。  
6. 对合并后的频域表示做逆 DFT（IDFT）恢复为时域令牌。  
7. 引入归一化正则项 $L_{\text{pen}} = \lambda \frac{\sum_k m_{b,k}}{\sum_k n_{b,k}}$，鼓励紧凑表示。

**提示令牌金字塔压缩（Prompt Token Compression）**  
1. 各层提示令牌数量按几何速率衰减：$P_{l+1} = \lceil 0.25 P_l \rceil$。  
2. 通过可学习的行归一化压缩矩阵 $W^{(l)}$，将当前层提示令牌压缩至下一层所需数量：$S^{(l)} = W^{(l)} H_P^{(l)}$。  
3. 同时提取全局残差摘要 $r^{(l)} = \text{MeanPool}(H_P^{(l)})$，并与压缩后的令牌拼接，作为下一层提示输入 $H_P^{(l+1)} = S^{(l)} + \mathbf{1}_{P_{l+1}} r^{(l)}$。

**理论保证（补充）**：作者给出稳定性分析，证明只要按能量阈值保留主导频谱分量、仅在局部相似令牌间合并，最终输出扰动在 LLM 主干和任务头满足 Lipschitz 连续条件下是可显式上界控制的。

## 6. 主要结论与发现

- **TS令牌存在显著频域冗余**：少数令牌在特定频带集中了大部分能量，大量令牌共享几乎无差别的频谱结构。
- **提示令牌影响呈金字塔衰减**：提示信息主要在浅层被TS令牌吸收，深层保留提示令牌几乎不带来边际增益，可由金字塔压缩大幅降低开销。
- **TokenDecouple 框架有效**：同时施加TS令牌频域合并和提示令牌逐层压缩，可在 78% 的评估实验中保持或提升性能，提供最高 7.68 倍推理加速。
- **自适应合并比率随任务/数据难度增加而下降**，说明方法能根据任务信息需求灵活调整压缩程度。
- **对比基线**：TokenDecouple 在性能保持度和加速比上均显著优于全局、局部、动态合并方法。

## 7. 优点

- **视角新颖**：从频域谱能量和信息可辨识性出发，对TS令牌进行非均匀、自适应压缩，跳出了传统时域分段合并的范式。
- **理论支撑**：对合并误差提供了基于 Parseval 等式、Whittle 似然和 Lipschitz 连续性的稳定性分析，增强了方法的可解释性。
- **解耦设计**：将TS令牌和提示令牌的压缩策略独立优化，更贴合两者的不同信息结构和作用机制。
- **实验扎实**：在多任务、多模型、多数据集上系统验证，并给出丰富的消融和敏感性分析。
