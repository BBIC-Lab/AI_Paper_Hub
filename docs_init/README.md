# AI Daily Paper Reader

欢迎使用 AI Daily Paper Reader。当前页面是初始化模板，不包含任何个人文献目录或密钥。

## 开始使用

1. 在首页完成 GitHub Token 与大模型配置。
2. 在后台添加你的订阅主题和检索查询。
3. 到 GitHub Actions 手动运行 `daily-paper-reader` 工作流。
4. 工作流完成后，新的日报会自动写入这里。
